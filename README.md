# 🤖 RVCE Admission Chatbot

A professional Full-Stack AI Chatbot designed to streamline the college admission process for **RVCE (Renganayagi Varatharaj College of Engineering)**.

## 🚀 Live Demo
Check out the live application here: [https://chatbot-lw0d.onrender.com](https://chatbot-lw0d.onrender.com)

## ✨ Features
* **Automated Admission Queries:** Instant answers for fees, courses, and eligibility.
* **Secure Login:** Protected access for students.
* **Smart UI:** Modern, responsive design for all devices.
* **Database Integration:** Real-time query handling using SQLite.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3 (Modern Glassmorphism)
* **Database:** SQLite3
* **Deployment:** Render

## 📁 Project Structure
* `app.py` - Main application logic.
* `templates/` - UI design files.
* `admissions.db` - Database containing admission data.
* `requirements.txt` - Required Python libraries.
